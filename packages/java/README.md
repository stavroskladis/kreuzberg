# kreuzberg - Java Bindings

High-performance document intelligence library

## Installation

Add to your `pom.xml`:

```xml
<dependency>
    <groupId>dev.kreuzberg</groupId>
    <artifactId>kreuzberg</artifactId>
    <version>4.8.5</version>
</dependency>
```
